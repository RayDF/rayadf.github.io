About Me

Saya adalah mahasiswa yang berkomitmen untuk membangun karir di dunia teknologi. Saat ini saya sedang mengejar gelar ahli madya pada program studi Teknik Informatika di Universitas Amikom Yogyakarta. Walaupun saya belum memiliki pengalaman profesional langsung, saya telah mendalami konsep-konsep dasar dan keterampilan yang diperlukan untuk menjadi seorang profesional yang sukses di bidang ini.

Dalam persiapan saya untuk memasuki dunia kerja, saya sedang fokus untuk memperdalam pemahaman saya tentang bahasa pemrograman. Diantara nya HTML, CSS, Javascript, dan PHP. Dan juga bebereapa framework seperti CodeIgniter, laravel dan Ionic. Saya juga berencana untuk mengikuti magang atau proyek kecil yang dapat memberi saya pengalaman praktis dalam dunia profesional.
